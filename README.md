# Controls

Draw:

* Left Mouse -> Draw
* Ctrl + Left Mouse -> Draw Line
* Right Mouse -> Erase
* 1 / 2 / 3 / 4 -> White / Red / Green / Cyan

Camera:

* Scroll -> Zoom
* Shift Scroll -> Zoom without changing brush size
* Ctrl Scroll -> Change brush size
* Middle Mouse or Space -> Pan

Other:
* Z -> Undo
* R -> Reset
